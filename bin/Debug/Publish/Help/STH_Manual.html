1. Introduction
The document provides an explanation about the Style Wise Report from the
basics. This document is intended to provide the knowledge that is necessary
for a user who has no knowledge about this system before.
It is highly recommended for the system users to go through the user guide
before they start using the system which will sort out the majority of their
problems.

2. Logging in to the system
Step 1:
Go to APMaS.lk and enter the login credentials
Step 2:
Search for MSP in the APMaS search bar
Step 3:
Click on the ‘Report View’
Page 2
Step 4:
Next, click on the ‘Other Reports’
Step 5:
Finally, the below screen will appear on your screen. Click on the ‘Style Wise
Report’ to proceed

3. Style Wise History
Shown below is a list of name of documents that could be viewed using the
Style Wise History page.
To begin with, the user must first enter the style number or else user can be
able to find the relevant style by using a DID . Afterwards the user must click
on the search button to load the results. Then the MSP history will be
generated as it is the first option in the report list. The fetched results of the
‘MSP History’ is discussed in the chapter 3.1.
Page 4
3.1 Style Wise History - MSP History
The below given table displays the MSP history for the style number entered
by the user. Listed below in the table is a log list which displays all the activities
that were carried on that particular style and/or DID number.
3.2 Style Wise History - Ticket History
Ticket history as shown in the below screenshot is designed to view all the
tickets that were/are placed under “MSP Data Removal” category for the
selected style number. The tickets which are pending, cancelled and
completed will all be displayed here.
Page 5
3.3 Style Wise History - Buyer PO
As visible in the below table, the Buyer PO report displays all the PO’s that are
made on that style number. The table includes data like PO Qty, POID,
extended ship date etc.
The PO’s made by the commercial department will be displayed here with the
use of the extended ship date.
Clicking on the “+” will drop down a detailed view of the PO with colour name,
colour code and the quantity.
Page 6
3.4 Style Wise History - MSP Schedule
The MSP Schedule displays a summary of all the shipments that were sent on
for the selected style number. Some of the information displayed are the FOB
Quantity, Status (Pending, Dispatched or Shipped) and the dispatched date.
3.5 Style Wise History - Size FOB and Invoice
This report displays a summary of Invoice details with the size wise fobs. The
reports includes data like the ship number (Invoice Number), FOB and the
shipped quantity etc.
Page 7
3.6 Style Wise History - Trash
The trash displays the records that were deleted from the MSP system. Below
given a screenshot of some items in the trash.This could be viewed by the
commercial department.
3.7 Style Wise History - Consumption Agreement
The consumption agreement displays all the revisions made against the
selected style number with the related details. Clicking on the select button on
the left side of the table will display an addition table which will display further
details on the selected record.
The second table as shown above displays a clickable link on the left side.
This will direct the user to a consumption guideline report as shown in the
below screenshot.

3.8 Style Wise History - Cancel
As the name suggests, this report displays all the shipments that were
cancelled & cancel requested by the user.

invoice. The user must first enter the invoice number and click on the ‘Search’

style wise report project as “Invoice Wise”
